В Python существует следующие типы данных:

Здесь приведены все типы данных, основные мы обязательно рассмотрим, двигаясь дальше по картотеке.
В папке "Дополнительно" содержится больше информации для Ваших интересов.

Числовые типы данных:
•int - целые числа 
•float - числа с плавающей точкой
•complex - комплексные числа

Типы данных для последовательностей:
•str - строки
•bytes - байтовые строки
•bytearray - изменяемые байтовые строки
•list - списки
•tuple - кортежи
•range - диапазоны чисел

Типы данных для множеств и отображений:
•set - множества
•frozenset - неизменяемые множества
•dict - словари

Логический тип данных:
•bool - логические значения True и False

Тип данных None:
•NoneType - единственный объект типа None

Кроме того, в Python есть множество встроенных функций и модулей для работы с различными типами данных и их преобразованиями.
