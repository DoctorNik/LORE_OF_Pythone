Множественное наследование, говорит нам о том, что класс может являться производным от одного и более родительских классов.
При множественном наследовании мы получим в свое распоряжение все функции от всех родительских классов.

EX.

class Base1():
  pass
  
class Base2():
  pass

class MultiDerived(Base1, Base2): # наследник обоих классов родителей: Base1 и Base2
  pass
  
Также возможен следующий варинат задания множественного наследования
EX.

class Base():
  pass
  
class Derived1(Base):
  pass

class Derived2(Derived1):
  pass
