Функция open() - предназначена для открытия файлов.


EX.

file = open("C:/text.txt")

Сущесвтуют разные режимы открытия, указываются 2-ым аргументом

	 r | Открывает файл для чтения
	 w | Открывает файл для записи. Создает новый файл, если он не сущ-ет, или полностью перезаписывает сущ-ющий
	 a | Открывает файл для добавления в конце файла. Создает новый файл, если он не сущ-ет
	 x | Открывает файл для эксклюзивного создания. Если файл уже сущ-ет, операция завершается неудачей
	 t | Открывает в текстовом режим (по умолчанию)
(r,w,a)b | Открывается в двоичном режиме
(r,w,a)+ | Открывает файл для обновления (Чтения и записи)

Нужно также учитывать кодировку, по умолчанию зависит от ОС. Для Windows = cp1251 | Linux = utf - 8

Функция close() - функция, закрывающая файл. После открытия файла и выполнения всех нужных операция, файл надо закрыть.

file = open("text.txt", "w", encoding = "utf-8")

file.close()


ЛУЧШИЙ МЕТОД для правильного закрытия файла - конструкция with. Автоматически закрывает файл

with open("text.txt", "w", encoding = "utf-8) as file:
	pass
